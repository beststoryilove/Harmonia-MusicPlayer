隐私条款

1. 数据收集与使用

Harmonia音乐播放器是一个纯静态网页项目，我们承诺：

  • 本项目不会主动收集您的任何个人信息
  • 所有用户数据（包括播放列表、我的收藏和播放历史）仅通过浏览器的本地存储(localStorage)保存，不会上传到任何服务器
  • Harmonia-桌面歌词功能的所有数据交换均通过WebSocket在内网环境下进行，仅传输歌词相关信息，不会泄露任何用户数据

2. 第三方服务

本项目使用了以下第三方API服务：

  • xxAPI（小小API）：提供4K背景支持
  • GDStudio（GD音乐台）API：提供音乐数据服务

请注意：

  • 我们无法保证这些第三方服务提供商不会收集您的个人信息
  • 我们也无法左右这些第三方服务提供商不会收集您的个人信息
  • 建议您查阅这些服务的隐私政策以了解其数据收集行为
  • 如果您不接受个人的信息被收集，则不建议您使用本项目

3. 托管平台

本项目托管在以下平台：

  • GitHub Pages：提供静态网页托管服务
  • CloudFlare：提供CDN加速服务

这些平台可能会收集访问日志等基本信息，我们无法控制这些数据的收集。

4. 数据安全

虽然我们采取了合理措施保护您的数据安全，但请注意：

  • 任何互联网传输都存在潜在风险
  • 浏览器本地存储的数据可能因浏览器设置或设备更换而丢失
  • 建议定期备份您的播放列表等重要数据
